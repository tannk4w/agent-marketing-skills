# Marketing Slack

Use this skill to prepare short Slack-ready marketing announcements, campaign updates, launch notes, and internal status messages.

## Inputs

Clarify or infer safely:

- Audience or channel.
- Purpose of the update.
- Key message.
- CTA or requested action.
- Tone.
- Timing or deadline.
- Links or assets to include.

## Workflow

1. Identify the key message and audience.
2. Keep the update short, scannable, and action-oriented.
3. Put the most important information first.
4. Include a clear CTA when needed.
5. Use bullets only when they improve readability.
6. Remove unsupported factual claims.

## Output Guidelines

- Prefer concise copy ready to paste into Slack.
- Include a subject-style opening line when helpful.
- Keep formatting simple.
- Add a brief note only if assumptions or missing context matter.

## Hard Rules

- Do not fabricate launch details, metrics, links, or deadlines.
- Do not use excessive hype unless the user asks for that tone.
- Do not default to Vietnamese from the user's chat language. Write in the language explicitly confirmed in the brief.
